# API Modules

This folder contains the domain modules for the multi-client automation platform:

- `auth`
- `clients`
- `whatsapp`
- `knowledge`
- `pricing`
- `templates`
- `leads`
- `meetings`
- `conversations`
- `quotations`
- `rag`
- `ai`

Each module should eventually expose its own router, service layer, persistence layer, and DTO/schema definitions.
